这是主引导程序的代码，但是此时还没有从硬盘读取core文件，就使用core中设定的选择子：
```asm
flush:                                  
       mov eax,0x0008                     ;加载数据段(0..4GB)选择子
       mov ds,eax
    
       mov eax,0x0018                     ;加载堆栈段选择子 
       mov ss,eax
       xor esp,esp                        ;堆栈指针 <- 0 

```
原因：
- 引导和内核之间的约定，这些是固定的
- 简化启动的流程
